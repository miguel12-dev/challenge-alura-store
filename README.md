# 📊 Análisis de Tiendas - Challenge Data Science (Alura Latam)

Este pequeño proyecto es parte de un **challenge del curso de Data Science de Alura Latam**, donde se trabaja con datos de ventas de **4 tiendas diferentes** para analizar su rendimiento y sacar algunas conclusiones útiles.

La idea del ejercicio es simular un caso sencillo de análisis de datos donde se comparan varias tiendas usando información como ventas, categorías de productos, calificaciones de clientes y costos de envío.

## 🧠 ¿Qué se analizó?

A partir de las bases de datos de las 4 tiendas se revisaron varias cosas importantes:

- **Ingresos totales de cada tienda**
- **Categorías de productos más vendidas**
- **Calificación promedio de los clientes**
- **Productos más y menos vendidos**
- **Costo promedio de envío**

También se generaron **algunos gráficos** para visualizar mejor la información, como por ejemplo ingresos por tienda, distribución de ventas por categoría y calificaciones de clientes.

## 📈 Resultados generales

Al revisar los datos se pueden ver algunas conclusiones interesantes:

- **La Tienda 1** es la que genera **más ingresos totales**.
- **La categoría de muebles y electrónicos** es la que más ventas tiene en casi todas las tiendas.
- **La Tienda 3** tiene la **mejor calificación promedio de clientes**.
- **La Tienda 4** tiene el **costo de envío promedio más bajo**.

## 🏪 Recomendación final

Después de comparar los datos, la recomendación sería considerar **vender en la Tienda 3**.

Aunque no es la que más ingresos genera, tiene algunos puntos fuertes:

- Tiene **la mejor calificación de clientes**.
- Tiene **muchas ventas en la categoría de muebles**, una de las más populares.
- El **costo de envío es razonable**, lo que puede ayudar a mantener a los clientes interesados.

Esto sugiere que puede ser una buena opción para obtener **buen rendimiento y clientes satisfechos**.
